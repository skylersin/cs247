Bases

Did you notice how we changed the height of the beakers? Each beaker can hold a different number of units, which changes depending on its base.

What is a base?

A base is the number of symbols that a system uses to represent numbers. Above, the beakers on the left are in base 6 because there are six symbols: 0, 1, 2, 3, 4, and 5.
