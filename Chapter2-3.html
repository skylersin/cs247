Place Values

Did you notice how each beaker held a different amount of liquid? We can think of the beakers as place values in math. Depending on their place among the other beakers, they have different values.

In base 10, the left beaker represents the hundred's place, so each unit in that beaker represents a value of 100. The far right beaker has the smallest value, 1 in base 10.

So how does that give us 103?

We add each together the product of the place value and the digit in its place.

(1 x 100) + (0 x 10) + (1 x 3) = 103
